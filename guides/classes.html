Class Path Guides

Beginning-to-end route maps for the major Orna and Hero of Aethric class families, written as practical planning advice rather than a tier list.

Last updated July 1, 2026Codex-based

How class paths work

Orna is flexible: unlocking a class does not permanently lock you into a tree. Think of these pages as natural progression lanes. You can buy side classes for skills, passives, or gear access, then return to your main route.

Tower Shard celestial classes by route

Celestial classes are the T10 follow-ups unlocked through Towers of Olympia progression and Tower Shards. Treat them as expensive endgame commitments: read the Towers guide and verify current costs in-game before spending.
